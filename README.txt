
# LiBee Website (Static)

- 5 pages: index, about, recruit, contact, privacy
- Minimal, responsive, dark-mode friendly
- Forms use `mailto:` (recommend switching to Google フォーム / Form.run / Netlify later)

## Quick Deploy

### GitHub Pages
1. このフォルダをそのまま GitHub の新規リポジトリにアップロード
2. Settings → Pages → Branch を `main` / `/ (root)` に設定

### Netlify
1. netlify.app で New site from Git
2. Build settings は不要（静的ファイル）

### ローカルで開く
`index.html` をブラウザで開くだけでOK
